Hardware Acceleration for Genomic Applications

We found that conventional architectures exploit low memory-level parallelism so that the available memory bandwidth cannot be fully utilized. Niubility accelerator customizes bit-wise operations and exploit data-level parallelism, that produces maximal concurrent memory accesses to saturate memory bandwidth. We implement an acceler- ator ASIC in a ST 28nm process that achieves up to 990x speedup over the state-of-the-art software..

Architecture and Application for Emerging Memory Technologies

We propose a practical, en- ergy efficient, Dual-Inline Memory Module (DIMM) based, NDP Accelerator for DNA Seeding Algorithm (MEDAL), which is based on off-the-shelf DRAM components. Experimental results show that MEDAL can achieve 21.14x/7.78x/3.94x speedup and 218.21x/6.80x/3.28x energy reduction when compared with a 16-thread CPU baseline and two state-of- the-art NDP accelerators, respectively.
